接口参数说明文档
===============
```
    接口域名: domain.com
    接口网关: gateway.do
    接口地址: https://domain.com/gateway.do?method={接口方法名}
```
## 账户体系接口

### 账号登陆
账号登录/注册允许使用

| 方式（type）         | 描述                             | 附加                       |
| ------------------- | ------------------------------- | ------------------------- |
| 常规(account)       | 使用账号或邮箱登录/注册              | 账号密码都传的情况下账号优先  |
| 手机(phone)         | 使用手机号登录/注册                 | 登录时需通过验证码验证       |
| 手机自动(autophone)  | 使用手机号登录/未注册用户自动注册并登录| 登录时需通过验证码验证       |
| 自选                | 可以通过接口设置使用以上任意方式登录   | 需传递对应 type            |
